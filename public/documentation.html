CineFan Movie App

Objective:

To build the server­side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

URL Endpoints/Request Methods

Request URL HTTP Method Response
Get a list of movies /movies GET A JSON object holding data about all movies
Get data about a single movie by title /movies/[TITLE] GET A JSON object holding data about a single movie
Get data about a genre by name /movies/genre/[NAME] GET A JSON object holding data about a genre of movies
Get data about a director by name /movies/directors/[NAME] GET A JSON object holding data about a director
Allow new users to register /users PUT A text message indicating whether the user was successfully added
Allow users to update their info (username) /users/[NAME]/info/[USERNAME] PUT A text message indicating the user and the updated username
Allow users to add a movie to their list of favorites /users/[NAME]/favorites/[TITLE] POST A text message indicating whether the movie was successfully added
Allow users to remove a movie from their list of favorites /users/[NAME]/favorites/[TITLE] DELETE A text message indicating whether the movie was successfully removed
Allow existing users to deregister /users DELETE A text message indicating whether the user was successfully removed
